Rysuje przycisk oraz jego cień na podanym ekranie. Tekst jest wyśrodkowany względem przycisku.

event_handeler(event, mouse)¶

Obsługuje zdarzenia myszy. Zmienia kolory przycisku po najechaniu kursorem. Zwraca True, gdy przycisk zostanie kliknięty i aktywowany.

Args:

event (pygame.event.Event): event taki jak kliknięcie myszką w obiekt mouse: (krotka złożona z dwych elementów - współrzędnych)pozycja myszki

Returns:

True jeżeli przycisk został kliknięty

update()¶

Uaktualnia stan przycisku. Po 4 sekundach od kliknięcia przycisk staje się znowu aktywny.
