class main.Game¶

Bases: object

run()¶

Główna pętla gry. Obsługuje: - zdarzenia użytkownika (np. zamknięcie okna), - logikę stanu gry i jego przełączanie, - interakcję sprite’ów z użytkownikiem, - aktualizację i rysowanie sprite’ów, - przejście do stanu końcowego gry, jeśli wszystkie postacie znikną.
